# Multi-Page-Restaurant-Website
Multi-page café ordering web app built with HTML and CSS featuring menu navigation and a simulated ordering system.

## Overview
This project is a multi-page café ordering website built using HTML and CSS. It simulates a simple online ordering experience where users can browse different menu categories and place a basic order through an interactive interface.

The website includes multiple linked pages to demonstrate navigation and structured front-end design.

## Pages Included
  Home – Landing page introducing the café concept
  Menu – Central menu page where users choose between drinks or breakfast options
  Order – Simulated ordering interface where selections are made
  About Us – Information about the café and its concept
  Sign Up – User registration page (simulated form UI)

## Features
- Multi-page navigation using HTML linking
- Centralized menu system with nested categories
- Scroll-based and section-based menu browsing
- Consistent UI design across all pages
- Basic interactive ordering flow
- Form-based sign-up page UI

## Tools Used
- Visual Studio Code
- HTML5
- CSS3
